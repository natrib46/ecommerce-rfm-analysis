# Análise Exploratória e Segmentação RFM: E-commerce Strategy

📌 Sobre o Projeto
Este projeto representa a intersecção entre Product Design e Marketing Data Analytics. Utilizando o framework de Métricas Piratas (AARRR), foquei no estágio de Retenção e Engajamento para transformar dados transacionais brutos em inteligência de negócio.

O objetivo foi processar um dataset de e-commerce, realizar a limpeza dos dados, calcular métricas de RFM (Recência, Frequência e Valor Monetário) e visualizar os resultados em um dashboard interativo para tomada de decisão estratégica.

🛠️ Tecnologias e Ferramentas
Linguagem: Python (Pandas, NumPy) para processamento e clusterização.

Design de Interface: Figma (Criação de background e UI do dashboard).

Data Viz: Looker Studio para visualização dinâmica.

Metodologia: Análise RFM (Recency, Frequency, Monetary).

📊 O Dashboard
https://lookerstudio.google.com/reporting/3d529a84-e0c4-40da-bb9b-9d86c67b5639

O dashboard utiliza um background personalizado criado no Figma para garantir uma experiência de usuário (UX) premium e foco na legibilidade dos KPIs.

📈 Metodologia RFM
Para esta análise, segmentamos a base de clientes em 4 perfis principais:

VIP / Potencial: Clientes de alto valor e alta frequência.

Fiel / Potencial: Clientes constantes com confiança estabelecida na marca.

Atenção / Promissor: Clientes recentes com potencial de fidelização.

Em Risco / Hibernando: Clientes que necessitam de campanhas urgentes de reativação.

🚀 Insights e Resultados
Identificamos que 40% da base é composta por clientes "Promissores", representando a maior oportunidade de crescimento de curto prazo.

A implementação de estratégias de upsell para os clientes "Fiéis" pode elevar o ticket médio em até [X]%.

O dashboard permite filtrar a performance por categoria, facilitando a identificação de produtos "estrelas".
